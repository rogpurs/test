# SKy Navia — SC LIFE

電子部品・半導体のグローバル調達ブランド **SKy Navia**（SC LIFE）の静的ウェブサイトです。

## ページ構成

| ページ | URL |
|---|---|
| トップページ | `/` |
| 会社概要 | `/about.html` |
| 在庫検索 | `/search.html` |
| 在庫買取 | `/purchase.html` |
| お問い合わせ | `/contact.html` |
| プライバシーポリシー | `/privacy-policy.html` |
| 利用規約 | `/terms.html` |
| サイトマップ | `/sitemap.html` |
| 404ページ | `/404.html` |

## ファイル構成

```
/
├── index.html
├── about.html
├── search.html
├── purchase.html
├── contact.html
├── privacy-policy.html
├── terms.html
├── sitemap.html
├── 404.html
├── assets/
│   ├── css/style.css       — グローバルCSS（デザイン変数含む）
│   └── js/main.js          — コンポーネントローダー・インタラクション
└── components/
    ├── header.html         — 共通ヘッダー
    └── footer.html         — 共通フッター
```
